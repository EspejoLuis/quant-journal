# Don't Outsource the Learning

## 📝 Summary

[Article](https://addyosmani.com/blog/dont-outsource-learning/) by [Addy Osmani](https://addyosmani.com/) about the danger of outsourcing learning to AI.

---

## 🔑 Key points

- AI is used by default for completing tasks but not to learn.

- Using AI (with no learning) creates **Cognitive Debt**: "more efficiency today (*less mental effort*) for less *critical thinking* in the future".

- How to fix it ? "Choose a workflow that does both shipping and teaching". Use "Learning Mode", ask why things have been done in a way, debug!
